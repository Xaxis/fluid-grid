Equal Width Columns

Basic equal width columns can be constructed using the fg-col() mixin by passing a single parameter defining the number of columns. Turning off the gutter between the columns is accomplished by passing $no-gutter: true. 

@include fg-col(1);
@include fg-col(2, $no-gutter: true);
@include fg-col(3);
@include fg-col(4, $no-gutter: true);
@include fg-col(5);
@include fg-col(6, $no-gutter: true);
@include fg-col(7);
@include fg-col(8, $no-gutter: true);
@include fg-col(9);

Variable Width Columns

Columns can also be created that span variable ratio dimensions based upon a 12 column grid.

@include fg-col(2, $mod: 1);
@include fg-col(2, $mod: 2, $no-gutter: true);
@include fg-col(2, $mod: 3)
@include fg-col(2, $mod: 4, $no-gutter: true);
@include fg-col(2, $mod: 5);
@include fg-col(2, $mod: 6, $no-gutter: true);
@include fg-col(2, $mod: 7);
@include fg-col(2, $mod: 8, $no-gutter: true);
@include fg-col(2, $mod: 9);
@include fg-col(2, $mod: 10, $no-gutter: true);
@include fg-col(2, $mod: 11);
@include fg-col(2, $mod: 12);

Multiple Variable Width Columns

Defining more than two variable width columns is also possible.

@include fg-col(3, $mod: 2);
@include fg-col(3, $mod: 8);
@include fg-col(3, $mod: 2);

@include fg-col(3, $mod: 2, $no-gutter: true);
@include fg-col(3, $mod: 8, $no-gutter: true);
@include fg-col(3, $mod: 2, $no-gutter: true);

@include fg-col(3, $mod: 3);
@include fg-col(3, $mod: 6);
@include fg-col(3, $mod: 3);

@include fg-col(3, $mod: 6, $no-gutter: true);
@include fg-col(3, $mod: 4, $no-gutter: true);
@include fg-col(3, $mod: 2, $no-gutter: true);
