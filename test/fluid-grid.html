fluid-grid

https://github.com/Xaxis/fluid-grid
Basic Columns

Basic columns can be constructed using the fg-col() mixin by passing a single parameter defining the number of columns to be created. All basic columns are the same width per row and fluid however they are not responsive unless the $breakpoint parameter is passed.

Responsive Columns

All column types can be enabled to respond at any dimension by specifying the width at which they should break. For instance to have a two column layout break at a "mobile" 720px you would use fg-col(2, 720px).

Variable Columns

Columns can also be created that span variable ratio dimensions.

Nested Columns

Nested columns are as easy as embedding fluid-grid styled elements within fluid-grid styled elements. There's no trick to them at all.

Ratio Columns

Ratio columns are no different than other fluid-grid columns other than you specify the fraction you would like a given column to span. They too can easily be made responsive by passing the $breakpoint parameter to fg-col()

fg-col($mod: '1/5')

fg-col($mod: '1/4')

fg-col($mod: '1/3')

fg-col($mod: '2/5')

fg-col($mod: '1/2')

fg-col($mod: '3/5')

fg-col($mod: '2/3')

fg-col($mod: '3/4')

fg-col($mod: '4/5')
